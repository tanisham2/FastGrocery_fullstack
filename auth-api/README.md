# Fast Grocery Backend API

Backend API for the Fast Grocery full-stack application.

Built using Node.js, Express.js, MongoDB, and JWT authentication.

## Features

* User Authentication & Authorization
* OTP-Based Password Reset
* Product and Cart Management
* Order Processing
* MongoDB Integration
* Swagger API Documentation


## Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT
* bcryptjs
* Nodemailer
* Swagger UI

## API Modules

The backend currently provides **17 REST APIs** across the following modules:

* Authentication APIs
* Product APIs
* Cart APIs
* Order APIs
* User APIs

## API Documentation

Swagger documentation is available at:

```text
http://localhost:5000/api-docs
```

## Run Locally

Install dependencies:

```bash
npm install
```

Start the server:

```bash
npm run dev
```

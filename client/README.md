# Fast Grocery Frontend

Frontend for the Fast Grocery full-stack application built using Next.js.

## Run Locally

```bash
npm install
npm run dev
```

Application runs at:

```text
http://localhost:3000
```

## Features

* User Authentication
* Product Listing
* Shopping Cart
* Order Management
* Responsive UI

Backend APIs are located in the `auth-api` folder of the root project.
